配置说明

  1. 访问 OpenAI Platform
  2. 注册或登录账户
  3. 进入 API Keys 页面
  4. 点击 "Create new secret key" 创建新密钥
  5. 复制密钥并粘贴到上方配置中
请妥善保管API密钥，不要泄露给他人

  • OpenAI API - GPT-3.5, GPT-4 系列
  • Claude API - Claude 3 系列
  • 自定义API - 兼容OpenAI格式的其他服务
推荐使用GPT-4或Claude 3 Sonnet以获得最佳效果
